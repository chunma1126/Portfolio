- ArrowPool을 활용하여 오브젝트를 효율적으로 관리했습니다.

- ArrowPattern을 추상 클래스로 구성하고, Pattern Queue를 만들어 다양한 패턴을 유연하게 조합할 수 있도록 구현했습니다.  
- Android Studio를 이용하여 빌드를 진행해 APK파일을 뽑아봤습니다.
